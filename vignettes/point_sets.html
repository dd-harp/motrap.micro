Point Sets

library(ramp.micro)

Introduction

In ramp.micro, resource point sets are stored in two columns, and there are utilities for creating point sets for toy models. This is a quick guide.

Uniform

unif_xy(n, mn, mx) generates random uniform point sets on a box:

  • mn \(< x <\) mx

  • mn \(< y <\) mx

The points are returned as a pair of named vectors â€“ the first column is named x and the next one y

p1 <- unif_xy(150, -10, 10)
head(p1, 4)
##                x         y
## [1,]  0.48740153 -5.464512
## [2,] -8.87803044 -4.184825
## [3,]  0.05414988 -7.403712
## [4,]  3.61318525 -6.946741

Note that p1 is \(150 \times 2\)

dim(p1)
## [1] 150   2
par(mar = c(2,2,1,1), bty = "o")
plot(p1)

Lattice

The function lattice(n, mn, mx) generates an \(n \times n\) lattice inside a box:

  • mn \(< x <\) mx

  • mn \(< y <\) mx

p2 <- lattice(17, -8, 8)
par(mar = c(2,2,1,1), bty = "o")
plot(p2)

dim(p2)
## [1] 289   2

Clusters

Two functions were developed to generate clusters. The first one, clusters_xy(xy, nc, vr) generates a poisson number of points (mean = nc) around a set of seeds. The distance from the center of each seed is a normal variate with variance vr.

seeds <- unif_xy(23, -20, 20)
p3 <- clusters_xy(seeds, 8, 1)
par(mar = c(2,2,1,1), bty = "o")
plot(p3)
